## 1. 产品简介

**FarmView-QR** 是一款面向数字农业基地的 IoT 监控展示平台。用户扫描基地专属二维码，即可免登录查看该基地的实时农业数据，包括：

- 农田地块边界地图与面积信息
- 气象站实时及历史气象数据
- 虫情监测设备数据
- 球机摄像头实时视频
- 农事行为任务记录
- 无人机农情监测记录

平台为移动端优先设计，适配微信扫码、iOS Safari、Android Chrome 等主流移动端环境。

## 2. 访问方式

### 2.1 扫码访问

每个基地对应一个专属二维码，扫码后自动跳转至该基地的监控页面，无需注册或登录。

### 2.2 URL 直接访问

```
http://服务器地址:3101/?site=基地标识
```

示例：
```
http://43.156.153.252:3101/?site=xinjiang-01
```

### 2.3 生成并分享二维码

在主页面点击右上角 **铃铛图标** → **基地推广** → 弹出当前基地的专属二维码：

- **手机**：长按二维码图片即可保存到相册
- **电脑**：点击「下载保存」按钮下载图片

## 3. 主页面功能详解

### 3.1 顶部栏（Header）

| 元素 | 位置 | 功能 |
|------|------|------|
| 头像 | 左上角 | 点击可选择系统预设头像（6款）或上传本地照片，自动保存 |
| 基地名称 | 顶部居中 | 默认显示基地名称；点击右侧铅笔图标可编辑，回车或点击勾选确认保存 |
| 气象标签 | 标题下方 | 显示当前天气状况（天气图标 + 温度 + 天气描述），数据来自 open-meteo |
| 铃铛图标 | 右上角 | 点击展开菜单，包含「基地推广」入口 |

> **提示**：头像和名称修改后自动保存在本地，下次访问时仍然保留。

---

### 3.2 地图展示区

地图默认展示基地内所有农田地块的边界轮廓，以及 IoT 设备的位置标记。

#### 农田地块

- 点击任意地块，底部弹出该地块的详情卡片，包含：地块名称、面积、作物种类、负责人等信息
- 地图默认自动缩放至合适视野，后续点击不会重置缩放级别

#### 地图全屏

- 点击地图右上角的 **双箭头图标** 进入全屏模式，地图铺满整个屏幕
- 再次点击（变为向内箭头）退出全屏

#### IoT 设备标记

地图上有三类设备图标：

| 图标颜色/样式 | 设备类型 |
|--------------|---------|
| 蓝色·气象图标 | 气象站 |
| 橙色·虫情图标 | 虫情监测仪 |
| 绿色·摄像头图标 | 球机摄像头 |

点击设备图标，右侧弹出该设备的详情面板。

---

### 3.3 IoT 设备弹窗

#### 气象站

显示气象站采集的历史环境数据，包含以下指标的折线图：

- 空气温度（°C）
- 空气湿度（%）
- 风速（m/s）
- 降水量（mm）

> 数据查询范围：最近 6 个月内，取非零有效值，并标注显示的日期范围。

#### 虫情监测仪

显示当前周期的虫情统计数据：

- 总诱虫数量
- 各虫种名称、数量及占比列表

#### 球机摄像头

- 自动拉取萤石云 HLS 实时视频流并播放
- 左上角显示红色「LIVE」标识（在线时闪烁）
- 右下角点击 **全屏图标** → 视频铺满全屏，右上角点击关闭按钮退出全屏
- 若设备离线，显示「设备离线」提示

---

### 3.4 农事行为区

展示基地内的农事任务记录，数据来源于农事管理系统。

#### 顶部统计概览

以彩色标签展示各状态任务数量：

| 状态 | 说明 |
|------|------|
| 未开始 | 计划中，尚未启动 |
| 进行中 | 当前正在执行 |
| 已完成 | 已结束的任务 |
| 未分配 | 尚未指派负责人 |
| 已取消 | 已废弃的任务 |

#### 任务列表

每条任务卡片包含：

- 任务名称与状态标签
- 优先级标签（紧急任务显示红色标签）
- 地块名称与作物种类
- 负责人
- 计划开始 / 结束时间

**默认展示最近 10 条**，点击底部「展开全部 N 条记录」显示所有任务。

#### 年份说明

系统根据当前月份自动选择默认年份：
- **1月～6月**：默认显示上一年的数据
- **7月～12月**：默认显示当年的数据

---

### 3.5 农情监测区

展示无人机巡田任务记录。

每条记录包含：

- 任务名称
- 所属地块
- 任务状态
- 飞行时间

**默认展示最近 10 条**，点击「展开全部」可查看所有记录。

> **说明**：当算法平台完成图像合成后，每条记录将自动显示无人机影像的彩色渲染图（rainbow 赋色）及 AI 农情分析文字和评级（优/良/中/差）。当前基地图像合成尚未执行，该功能待对接方补充数据后自动生效。

## 4. 申报基地表单

点击「我要加入」或「我要申报基地」后弹出申报表单。

### 填写字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 联系人姓名 | 文本输入 | 必填，请填写真实姓名 |
| 省 | 下拉选择 | 全国所有省/直辖市/自治区 |
| 市 | 下拉选择 | 选择省后自动加载对应地级市 |
| 县/区 | 下拉选择 | 选择市后自动加载对应县区（可选） |
| 详细地址 | 文本输入 | 必填，填写镇/村及具体地点 |
| 土地面积 | 数字输入 | 必填，单位：亩 |
| 联系电话 | 手机号 | 必填，需为有效的 11 位手机号 |

### 提交流程

1. 填写完所有必填项后，点击「**提交申请**」
2. 手机号格式验证通过后，数据提交至服务端数据库
3. 提交成功后显示「🌱 提交成功！」页面
4. 点击「关闭」退出表单

### 其他联系方式

表单底部提供直拨电话：**180-1010-0298**

## 6. 异常页面说明

当访问链接中的基地标识（`?site=xxx`）不存在或错误时，系统显示**兜底提示页面**：

- 提示用户当前访问的基地标识无效
- 显示当前已配置的所有基地链接（可点击直接跳转）
- 提供「**我要申报基地**」按钮，引导用户填写申报表单

## 7. 多基地管理说明

### 基地配置

每个基地在服务端 `sites-config.json` 文件中独立配置，字段包括：

```json
{
  "siteKey": "基地唯一标识",
  "siteName": "基地展示名称",
  "baseId": 数字ID,
  "apiAuth": {
    "username": "账号",
    "password": "密码"
  },
  "farmlandIds": ["地块ID1", "地块ID2", "..."],
  "devices": {
    "weatherIds": ["气象站设备ID"],
    "insectIds": ["虫情仪设备ID"],
    "cameraIds": ["摄像头ID1", "摄像头ID2"]
  }
}
```

### 新增基地流程

1. 向对接方获取新基地的 `baseId`、账号密码、地块 ID 列表、设备 ID 列表
2. 在 `sites-config.json` 中新增一条配置
3. 重启 Docker 服务使配置生效
4. 访问 `/?site=新基地标识` 验证数据正常
5. 通过主页铃铛菜单生成并分发该基地的专属二维码

### 访问地址格式

```
http://43.156.153.252:3101/?site=基地标识
```

## 8. 常见问题

**Q：扫码后页面一直显示「正在校验站点...」？**
A：服务器网络异常或容器未运行，联系管理员检查服务状态。

**Q：地图加载不出来？**
A：高德地图 API Key 可能失效，或网络无法访��高德服务。

**Q：气象数据显示异常或为空？**
A：open-meteo 接口偶发超时，刷新页面重试；若持续异常检查服务器出口网络。

**Q：摄像头显示「设备离线」？**
A：萤石云设备处于离线状态，确认摄像头硬件正常工作并已联网。

**Q：申报表单提交后管理员没有收到？**
A：数据已保存至服务器数据库，管理员登录 `/admin` 页面刷新即可查看。

**Q：如何备份申报数据？**
A：SQLite 数据库文件存储于 Docker volume `farmview_data`，路径为容器内 `/app/data/join_requests.db`。可通过 `docker cp` 命令导出备份，或直接在管理员页面导出 PDF。
